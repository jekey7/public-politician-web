# 공개 데이터 기준

이 문서는 MVP에서 화면과 스냅샷에 올릴 수 있는 정보의 경계를 정한다.

## 원칙

- 공개 대상은 공인의 공적 역할과 직접 관련된 정보로 제한한다.
- 모든 사실 값은 `source_url`, `source_org`, `fetched_at`, `license_note`가 있는 출처 메타데이터와 함께 보관한다.
- 출처가 없거나 라이선스 확인이 불가능한 값은 화면과 공개 스냅샷에 포함하지 않는다.
- 출처 간 값이 다르면 하나를 고르지 않고 불일치로 표시한다.
- 뉴스와 영상은 제목, 발행처, 발행일, 원문 링크만 보관한다. 원문 본문, 이미지, 썸네일 파일은 재호스팅하지 않는다.

## 허용 범위

- 국회의원 신분, 정당, 지역구, 직위처럼 공적 직무를 설명하는 정보.
- 공개 프로필 또는 선거/국회 공개자료에 등재된 학력, 경력, 정당 이력, 선거 이력.
- 공개 의정활동 자료에 포함된 발의 법안, 표결, 위원회 활동.
- 인물 관련 뉴스/영상의 메타데이터와 원문 링크.

## 제외 범위

- 공개 출처로 확인되지 않는 가족, 주소, 연락처, 재산 세부 항목, 건강, 종교, 사생활 정보.
- 정치적 평가, 성향 점수, 호감도, 평판, 의혹 단정처럼 사실 검증 범위를 벗어나는 정보.
- AI가 원문을 요약하거나 해석해 새 사실처럼 만든 문장.
- 원문 본문, 이미지, 썸네일 파일처럼 타인의 콘텐츠를 복제 저장하는 데이터.

## ASSUMPTION

- 실제 소스별 약관과 라이선스 검토 전까지 공개 스냅샷은 목 데이터 또는 내부 검증용 산출물로만 취급한다.
- 성별, 생년 등 인적사항은 공개 공식 프로필에 포함된 경우에도 MVP에서는 표시 필요성을 재검토한다.
