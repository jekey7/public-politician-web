# 대한민국 국회의원 정보 플랫폼

> AI 교차검증 기반의, 검증 가능한 객관성을 갖춘 비영리 현직 국회의원 정보 플랫폼

공개된 출처에 근거한 정보만을 다루며, AI를 활용해 여러 출처 간 정보의 불일치를 자동으로 탐지·표시합니다.

## 핵심 원칙

| 원칙 | 내용 |
|------|------|
| **객관성** | 주관·의견·평가를 일절 배제. AI는 사실을 생성하지 않으며, 공개된 출처 정보만 전달 |
| **투명성** | 코드·데이터·출처를 전부 공개하여 객관성을 주장이 아니라 증명으로 담보 |
| **검증** | 복수 출처의 정보 불일치를 AI가 드러냄. 어느 정보가 옳은지에 대한 판단은 이용자에게 맡김 |

기존 정치인 정보 서비스가 여러 출처를 **하나로 병합**하는 것과 달리, 이 플랫폼은 병합 대신 **출처 간 불일치를 드러냅니다.**

## 기술 스택

- **Frontend:** Next.js 15 (React 19), 정적 생성(SSG)
- **AI:** LLM (개체 정합·불일치 분류) + RAG (출처 인용 응답)
- **데이터 소스:** 열린국회정보 API, 공공데이터포털, 헌정회, 선거관리위원회
- **언어:** TypeScript

## 시작하기

```bash
npm install
npm run dev       # 개발 서버 (http://localhost:3000)
npm run build     # 정적 빌드
npm test          # 타입 체크 + 테스트
npm run lint      # 린트
```

### 스냅샷 생성

```bash
npm run snapshot:mock   # 목 데이터로 스냅샷 생성
npm run snapshot        # 실 데이터로 스냅샷 생성 (API 키 필요)
```

### 전체 검증

```bash
npm run verify:all      # 빌드·스냅샷·소스·라이선스·UI·접근성 등 전체 검증
npm run verify:release  # verify:all + npm audit
```

## 데이터 구조

성격이 다른 데이터를 분리합니다.

- **교차검증 데이터** — 인적사항, 의정활동. 복수 출처 수집 → AI 개체 정합 → 불일치 탐지·표시 → 출처 원문 링크
- **뉴스 피드** — 인물 관련 기사·영상. 검증 없이 시간순 나열, AI 요약 없음, 메타데이터+링크만 제공

## AI 역할 (3가지로 한정)

| 역할 | 내용 |
|------|------|
| 개체 정합 | 출처별로 다른 표기가 같은 항목을 가리키는지 의미적으로 판단 |
| 불일치 분류 | 표기 차이 / 내용 충돌 / 정보 누락을 구분 |
| 자연어 질의 (RAG) | 충돌 지점과 출처를 인용해 응답. 출처 없으면 "자료 없음" 반환 |
